# Morse Converter (Go)

Веб-приложение на Go для автоматической конвертации текста ↔ азбуки Морзе.

Приложение принимает файл через HTML-форму, определяет его содержимое  
(обычный текст или код Морзе) и возвращает сконвертированный результат.

---

##  Возможности

- HTTP-сервер на Go
- Загрузка файлов через HTML-форму
- Автоматическое определение:
  - текст → азбука Морзе
  - азбука Морзе → текст
- Поддержка русского алфавита
- Сохранение результата в локальный файл
- Простая и расширяемая архитектура

---

##  Используемые технологии

- Go
- net/http
- Работа с файлами (`os`, `io`)
- HTML-формы (multipart/form-data)
- Стандартный HTTP-роутинг
- Работа со временем (`time`)
- Структурирование проекта по пакетам

---

##  Структура проекта

morse-converter/

##  Запуск проекта

###  Перейти в корень проекта

- Убедись, что ты находишься в директории, где лежат `go.mod` и `index.html`.

###  Запуск сервера


- go run ./cmd

### После запуска сервер будет доступен по адресу:

- http://localhost:8080

### Проверка работы

### Открой в браузере:

- http://localhost:8080


### Загрузи файл:

- sample_morse.txt — содержит код Морзе
- sample_text.txt — содержит обычный текст

### Приложение автоматически:

- определит формат входных данных
- выполнит конвертацию
- вернёт результат в ответе
- сохранит результат в локальный файл с уникальным именем

### Архитектура

- Проект построен с разделением ответственности:
- handlers — обработка HTTP-запросов и формы загрузки файла
- service — автоопределение формата и запуск конвертации
- server — настройка HTTP-сервера (роуты, таймауты, логгер)
- pkg/morse — пакет конвертации текста и кода Морзе
